🛠️ Guia de Instalação: Moto G84 (bangkk)

Aviso: Este processo apagará todos os seus dados. Faça backup antes!

1. Preparação do Ambiente

  • Instale os Drivers da Motorola no seu notebook.
  • Baixe a pasta Platform-Tools (ADB/Fastboot).
  • Certifique-se de que o Bootloader do seu G84 está desbloqueado.

2. Organizando os Arquivos (IMPORTANTE)

Mova TODOS os arquivos da ROM que você baixou para dentro da pasta platform-tools. Os comandos só funcionam se os arquivos da ROM e os arquivos do ADB estiverem no mesmo local.

3. Modo Fastboot

Desligue o celular e ligue segurando Power + Volume Down. Conecte ao notebook via USB.

4. Comandos no CMD

Dentro da pasta platform-tools, digite cmd na barra de endereços do Windows para abrir o terminal diretamente lá. Digite os comandos na ordem:

fastboot devices
fastboot flash boot boot.img
fastboot flash dtbo dtbo.img
fastboot flash vendor_boot vendor_boot.img

5. Finalização

No celular, use os botões de volume para selecionar Recovery Mode, entre e selecione Factory Reset / Wipe Data. Depois, selecione Reboot System Now.
